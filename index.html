GitHub Pagesの練習用

github pagesを利用しクラウドでのホスティングの理解を深める。

クラウドホスティングとは

クラウドホスティングとは、インターネットを通じてサーバーリソースを提供するサービスです。

これにより、ユーザーは物理的なサーバーを所有せずに、必要なときに必要なだけのリソースを利用できます。

クラウドホスティングは、スケーラビリティ、柔軟性、コスト効率の面で多くの利点があります。

GitHub Pagesとは

GitHub Pagesは、GitHubが提供する静的ウェブサイトホスティングサービスです。

ユーザーはGitHubリポジトリにHTML、CSS、JavaScriptファイルを配置することで、簡単にウェブサイトを公開できます。

GitHub Pagesは、個人のポートフォリオサイトやプロジェクトのドキュメントサイトなどに広く利用されています。
